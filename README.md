# MNIST Digit Recognizer — ANN Model

A handwritten digit classifier built using sklearn's MLPClassifier on the MNIST dataset.

## Results
- Training Accuracy: 97.70%
- Predicted first test digit: 2 ✅

## Tech Stack
- Python 3.13
- pandas, numpy, matplotlib
- scikit-learn (MLPClassifier)

## How to Run
1. Clone the repo
2. Create virtual env: `python -m venv myenv`
3. Activate: `source myenv/bin/activate`
4. Install: `pip install -r requirements.txt`
5. Add dataset inside `datasets/mnist/`
6. Run: `jupyter notebook main.ipynb`

## Dataset
Download from [Kaggle - Digit Recognizer](https://www.kaggle.com/competitions/digit-recognizer/data)
